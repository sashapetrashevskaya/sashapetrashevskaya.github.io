Итого:

static

иначе называется "без розиционнирования"

relative

сдвигает элемент относительно текущего места

  1. противоположеные границы left/right(top/ottom)одновременно указать нельзя.
  2. Окружающие элементы ведут себя так,будто элемент никуда не сдвигался.

absolute

Визуально переносит элемент на новое место.
Новое место вычисляется по координатам left/right,top/bottom относительно ближайшего позиционированого родителя. Если такого родителя нет,то им считается окно.

  1. Ширина элемента по умолчанию устанавливается по содержимому.
  2. Можно указать одновременно противоположные границы left/right (top/bottom).Элемент при этом растянется.
  3. Окружающие элементы заполняют освободившееся место.

fixed

Повод абсолютного позиционирования,при котором элемент призываться к координатам окна,а не документа. При прокруте остается на том же месте.
